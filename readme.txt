== Description ==

**Multi Currency Pro for WooCommerce** is multi currency plugin for woocommerce, that allows your customers to view product prices in different currencies. It also allows store managers to enable selling products in different currencies.

#### Features

* Add unlimited currencies to your woocommerce store.
* Design currency sidebar from plugin settings page.
* Display currencies based on customer's location.
* Display currency sidebar on woocommerce page.
* Display price switcher on single product page.
* Enable customers to purchase products in currency of their choice.

== Installation ==

Install Multi Currency Pro for WooCommerce from the 'Plugins' section in your dashboard (Plugins > Add New > Search for Multi Currency Pro for WooCommerce ).

Place the downloaded plugin directory into your wordpress plugin directory.

Activate it through the 'Plugins' section.
